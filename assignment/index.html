Arrays

What is an array?

An array is a special variable, which can hold more than one value. An array can hold many values under a single name, and can be access using the index number. In array the first item/element starts at the 0 index.

Creating an array

Declare a variable name and assign it to a square bracket [ ]. Inside the [ ] is where you will list the elements separated by a comma. Make sure to still follow primitive data type rules.

Some useful array methods

  • length - tells you how many items are in the array
  • length -1; that -1 allow you to access the last element on array list
  • push(); the push method add a new element/item to the end of an array list
  • pop(); the pop method remove the last element/item from the array list
  • shift(); the shift method remove element/item from the beginning of the array list
  • unshift(); the unshift method add element/item to the beginning of the array list
  • splice(); the splice method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place
  • slice(); the slice method returns selected elements in an array, as a new array, however it does not change the original array
  • sort(); the sort method organize your data in order of alphabetically or numerically.
  • reverse(); the reverse() method reverses an array in place and returns the reference to the same array, the first array element now becoming the last, and the last array element becoming the first
  • toString(); the toString method returns a string as a string, it does not change the original and it can be use to covert a string object into a string.
  • join(); the join method returns an array as a string, it does not change the original string, and any separator can be specified, otherwise the default is comma
